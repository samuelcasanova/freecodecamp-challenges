## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:samuelcasanova/freecodecamp-challenges.git
   ```
2. Install the npm dependencies
   ```sh
   npm install
   ```
3. Now you can run the vitest tests
   ```sh
   npm test
   ```
